## The Short Version

I am Hooman Hajimohamadi, a Tehran-based full-stack developer with a frontend heart. I build polished web products with React, Next.js, TypeScript, Laravel, and TailwindCSS.

My favorite kind of work sits between design taste and engineering discipline: clean component systems, smooth product flows, fast pages, thoughtful animations, and code that the next developer can actually enjoy reading.
